## 📋 О проекте
- Доступ по API-ключу
- Локальное хранение заметок
- Автосохранение
- Экспорт в JSON
- Полностью офлайн
- Адаптивный дизайн

## 🚀 Быстрый старт
1. Откройте `index.html` в браузере
2. Используйте демо-ключ: `sk_demo_notes_2024`
3. Создавайте и редактируйте заметки

## 🔧 Управление ключами
- Откройте `admin.html` для добавления/удаления API-ключей
- Ключи хранятся в localStorage браузера

## ⚠️ Важно
**Не храните пароли, банковские данные и другую критическую информацию!**
Приложение предназначено для обычных заметок и черновиков.
